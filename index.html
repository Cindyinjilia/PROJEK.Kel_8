Limit Fungsi Dua Peubah

Limit fungsi dua peubah adalah konsep limit yang melibatkan dua variabel independen. Diberikan suatu fungsi dua peubah, \(z = f(x,y)\). Definisi Diberikan suatu fungsi dua peubah, sebutlah \(z = f(x,y)\). Definisi

\[ \lim_{(x,y)\to(x_0,y_0)} f(x,y) = L \]

apabila untuk setiap \(ε > 0\) terdapat \(\delta > 0\) sedemikian sehingga

\[ 0 < \|(x,y) - (x_0,y_0)\| < \delta \Rightarrow |f(x,y)-L| < ε. \]

Teorema 8.1.1

(Teorema Substitusi): Jika f(x,y) merupakan polinom dalam x dan y, yakni \[ f(x,y) = \sum_{i=0}^n \sum_{j=0}^m c_{ij} x^i y^j \]

maka \[ \lim_{(x,y)\to(a,b)} f(x,y) = f(a,b) \] Jika \[f(x,y) = \frac{p(x,y)}{q(x,y)}\] dengan \(p\) dan \(q\) polinom dalam x dan y, maka \[ \lim_{(x,y)\to(a,b)} f(x,y) = \frac{p(a,b)}{q(a,b)} \] Asalkan \(q(a,b) ≠ 0.\)

Teorema 8.1.2 (Teorema Apit).

Jika \(f(x,y) ≤ g(x,y) ≤ h(x, y)\) untuk \(xy ≈ x_0, y_0\) dan \[ \lim_{(x,y) \to (x_0, y_0)} f(x,y) = \lim_{(x,y) \to (x_0, y_0)} h(x) = L \] Maka, \[ \lim_{(x,y) \to (x_0, y_0)} g(x,y) = L \]

Kekontinuan

Kekontinuan adalah konsep dalam kalkulus yang menggambarkan sifat suatu fungsi di mana perubahan kecil pada input menghasilkan perubahan kecil pada output, tanpa adanya loncatan atau gangguan. Fungsi \(f(x,y)\) disebut kontinu di titik (a,b) jika limit \(f(x,y)\) ketika \((x,y)\) mendekati (a,b) sama dengan \(f(a,b)\). Jika tidak, fungsi dikatakan tidak kontinu di titik tersebut. Fungsi \(f(x,y)\) dikatakan kontinu di \((a,b)\) apabila

\[ \lim_{(x,y)\to(a,b)} f(x,y) = f(a,b) \].
